## ⚡ O Projeto
O objetivo aqui foi criar um painel "estilo gamer" com foco em **Dark Mode** e performance de navegação.

<img width="592" height="421" alt="image" src="https://github.com/user-attachments/assets/2db0e109-869d-48c4-b069-32d97c499ed6" />


### 🎨 Destaques da Interface:
- **Navegação Dinâmica:** Troca de abas instantânea usando `pack_forget()` para evitar consumo de memória.
- **Pasta de Assets:** Organização profissional de ícones e imagens.
- **Customização de Widgets:** Checkboxes e botões estilizados com as cores `#6A0E86` e `#8F108F`.
